# vue_shop
一个电商后台管理系统的前端项目，基于Vue+Element实现。主要包括用户管理、权限管理、商品管理、订单管理、数据统计等功能

## 技术栈

vue2.x + vue-cli + vue-router + element ui + es6 + less + flex + axios

## 开启后台api
运行 vue_api_server 中的 app.js
```
node app.js
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve

```
npm run build
